Julian Selley's Python Proteomics Library

This Python library, written for Python 2.7, provides classes and methods to support Proteomics data interpretation. It is intended to be a library resource for use by other software programmes, rather than providing any distinct program functionality. There are unittest modules to test the functionality of the code, and these (as well as the API documentation) should provide an example for implementation.

About

This Python package provides a number of modules which facilitate the interpretation of Proteomics data.
